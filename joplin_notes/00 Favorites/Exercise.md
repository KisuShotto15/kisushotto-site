Here's the best bang-for-your-buck bodyweight move per muscle group:

**Chest** → Push-ups (diamond variation once standard gets easy)
**Back** → Pull-ups (you have the bar — nothing beats it)
**Shoulders** → Pike push-ups (feet elevated for more load)
**Biceps** → Chin-ups (supinated grip, slow eccentric)
**Triceps** → Diamond push-ups (or dips if you have parallel surfaces)
**Quads** → Bulgarian split squats (rear foot on chair)
**Hamstrings/Glutes** → Nordic curl (anchor feet under something heavy) — if not possible, single-leg glute bridge
**Core** → Ab wheel rollout (or bodysaw with socks on smooth floor)
**Calves** → Single-leg calf raise on a step (full ROM)

---

**Now, the routine — real talk first:**

Training to failure *every single day* on the same muscles will wreck your recovery. You'll plateau in 2-3 weeks and start losing strength. The fix: **push/pull split alternating daily** so each muscle gets hit every 48h. Still daily training, but smart.

---

**Day A — Push + Legs**

| Exercise | Sets × Reps |
|---|---|
| Diamond push-ups | 3 × to failure (or 1-2 RIR) |
| Pike push-ups (feet elevated) | 3 × to failure |
| Bulgarian split squats (each leg) | 3 × to failure |
| Single-leg calf raise | 3 × to failure |

**Day B — Pull + Core**

| Exercise | Sets × Reps |
|---|---|
| Pull-ups | 3 × to failure (or 1-2 RIR) |
| Chin-ups | 3 × to failure |
| Nordic curls | 3 × to failure |
| Ab wheel rollout (or bodysaw) | 3 × to failure |

**Schedule:** A-B-A-B-A-B-A, repeat. Every day, alternating. Rest 2-3 min between sets.

**Progression rule:** Once you hit 15+ reps on any set, make the exercise harder (elevate feet, slow the tempo to 3s eccentric, add a pause at the bottom). Failure at 6-12 reps is the hypertrophy sweet spot — if you're repping 20+, you're training endurance, not muscle growth.
